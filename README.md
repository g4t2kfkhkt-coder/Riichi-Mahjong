# Riichi Mahjong

这是一个纯前端的日麻（雀魂规则风格）听牌与点数计算器。

## 项目用途

- 录入手牌、场风、自风、和牌方式等信息
- 计算当前是否听牌，以及可以和哪些牌
- 显示每张和牌的剩余枚数
- 计算役种、番数、符数、点数
- 支持副露、暗杠、宝牌、里宝牌、赤宝牌
- 支持本场数、供托数（立直棒）
- 支持双立直、天和、地和等常见特殊状态
- 区分“有役可和”与“形式听牌（无役不能和）”
- 自动保存最近一次录入内容，刷新页面后可恢复

## 当前状态

- 计分、符数、役种和特殊状态逻辑已完成一轮集中修复
- `start.bat` 会启动本地 PowerShell 静态服务并自动打开浏览器
- 项目运行不依赖 Python、`venv` 或其他后端环境

## 项目结构

- `index.html`：页面界面和全部核心计算逻辑
- `svg/`：麻将牌 SVG 图片资源
- `start.bat`：一键启动本地静态服务并自动打开浏览器
- `serve.ps1`：本地静态文件服务脚本

## 运行方式

1. 双击 `start.bat`
2. 浏览器会自动打开 `http://127.0.0.1:8765/index.html`

## 自动化测试

- 运行 `node scripts/run_scoring_tests.js`
- 当前脚本覆盖 26 组番数/符数回归样例，包含副露固定分解、三暗刻、七对子/二杯口择优、明暗杠符数、平和、七对子、三色、一气、纯全/混全、混一色/清一色、混老头、三杠子等边界

## 本地保存

- 页面会通过浏览器 `localStorage` 自动保存最近一次录入状态
- 刷新后会自动恢复手牌、副露、暗杠、宝牌、里宝牌、场风、自风、和牌方式、特殊状态、本场数和供托数
- 数据仅保存在当前浏览器本地，不会上传到外部服务

## 停止方式

关闭启动后弹出的 PowerShell 服务窗口即可。

## 说明

项目是纯前端静态页面，所有核心逻辑都在浏览器中运行。
